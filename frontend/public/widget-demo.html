üöÄ Quick Start

Add this code to your website, right before the closing </body> tag:

<!-- AgentForge Widget -->
<script src="https://your-domain.com/agentforge-widget.js"></script>
<script>
  AgentForge.init({
    apiKey: 'af_xxxxxxxxxxxxx',
    apiUrl: 'https://your-api.com/api/v1/public',
    position: 'bottom-right',
    primaryColor: '#3B82F6',
    title: 'Chat with us!',
    subtitle: 'We typically reply in a few minutes'
  });
</script>
Note: Replace af_xxxxxxxxxxxxx with your actual API key. You can generate API keys from your agent's dashboard.

AgentForge.init({

  placeholder: 'Type a message...', // Input placeholder
  avatar: 'https://...',           // Avatar URL (optional)
  zIndex: 9999                     // CSS z-index
});

üìù How to Get Your API Key

  1. Go to your AgentForge dashboard
  2. Select your agent
  3. Click on "API Keys" in the sidebar
  4. Click "Generate New API Key"
  5. Give it a name (e.g., "Website Widget")
  6. Copy the generated key and use it in your widget code

üîí Security Best Practices

  • Only use API keys on domains you trust
  • Set allowed origins in your API key settings to restrict usage
  • Monitor your API usage in the dashboard
  • Rotate API keys periodically
  • Disable unused API keys immediately
